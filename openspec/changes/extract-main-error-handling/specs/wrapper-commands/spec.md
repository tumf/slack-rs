## MODIFIED Requirements

### Requirement: エラー終了処理の内部共通化後もコマンド終了規約は維持される
`main` 内のエラー終了処理を内部ヘルパーへ共通化した後も、既存コマンドの標準エラー出力と終了コード規約（通常 1、非対話エラー 2）は変化してはならない。(MUST)

#### Scenario: 共通化後も非対話エラー終了コードが維持される
- Given 非対話エラーを返す write 系コマンドを実行する
- When エラー終了処理が共通ヘルパー経由で実行される
- Then 標準エラーに従来同等の失敗メッセージが出力される
- And 終了コードは 2 である
