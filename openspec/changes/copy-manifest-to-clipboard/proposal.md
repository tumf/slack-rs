# プロポーザル: auth login マニフェストのクリップボードコピー

## 背景
現在は `auth login` 実行時に Slack App Manifest を生成してファイルに保存するのみで、ユーザーが手動で開いてコピーする必要がある。

## 目的
マニフェスト生成時に YAML を自動でクリップボードへコピーし、Slack の「From an app manifest」画面に即貼り付けできるようにする。

## スコープ
- `auth login` 実行中のマニフェスト生成処理に、クリップボードコピーを追加する
- クリップボード操作失敗時は警告を表示し、処理は継続する

## スコープ外
- クリップボードの内容検証（外部アプリでの貼り付け確認）
- GUI 変更や追加の対話 UI

## 依存関係・リスク
- OS クリップボードへのアクセスは環境依存のため、失敗時の非致命処理が必要
