# 変更提案: OAuthスコープのプリセット対応

## 背景
OAuthログイン時のスコープ入力は現在、手入力または既定値の利用のみであり、ユーザーが「必要なスコープを全部入れて」と要求した際に簡単に一括指定できません。

## 目的
- 対話入力およびCLI引数でプリセット名 `all` を指定できるようにする
- `all` が包括的なスコープセットへ展開されることを保証する

## 非目的
- Slackアプリの権限を自動検出すること
- 管理者/Enterprise限定スコープの網羅

## 変更範囲
- `auth login` の対話入力
- `config oauth set --scopes` の入力解析
- ヘルプ/ガイダンス文言

## 受け入れ基準
- `all` (大小文字を問わず) が包括的なスコープ一覧へ展開される
- `all` と個別スコープが混在した場合、展開後に重複を除去して保存される
- 既定値の挙動は維持される（空入力時は既定スコープ）
