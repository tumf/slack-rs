Build: `cargo build`, `cargo build --release`.
Test: `cargo test`, `cargo test --verbose`, `cargo test <name>`, `cargo test --test <file>`.
Lint/format: `cargo fmt`, `cargo fmt -- --check`, `cargo clippy`, `cargo clippy -- -D warnings`.
Check/docs: `cargo check`, `cargo doc --open`, `cargo clean`, `cargo audit`.
Utility (Darwin): `git`, `ls`, `cd`, `find`, `rg`.
